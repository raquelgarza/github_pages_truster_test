Introduction

31 Jan 2023, 00:00

trusTEr - A trusting TE cluster analysis Version 0.1.1, written in Python 3.6.6 Takes fastq files from 10x single cell RNA sequencing, clusters cells using Seurat, and can be used to produce read count matrices in a cluster level. You can also quantify reads per cluster having predefined clusters. Requirements TrusTEr depends on several external software. We provide a Docker container and a conda environment for a quick-start.
